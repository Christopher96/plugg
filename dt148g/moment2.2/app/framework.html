Twitter Bootstrap

Twitter Bootstrap är ett ramverk som är perfekt för att skapa responsiva mobile first hemsidor. Bootstrap är ett opensource ramverk för utveckling inom HTML, CSS och JavaScript. Det stödjer flera Sass variabler och mixins, ett responsivt gridsystem, ett stort utbud av komponenter och användbara plugins för jQuery som gör utvecklingen snabb och effektiv.

Komponenter

Det finns många färdiga komponenter att välja på i bootstrap. Genom att använda CSS klasser som .input-group, .navbar, .alert eller .dropdown så kan man snabbt skapa responsiva komponenter som fungerar anpassar sig till storleken. Om man vill ändra utséende så kan man använda ex. .bg-primary eller .navbar-dark och sen ändra Sass variablerna för att få ett enhetligt tema.

Gridsystem

Bootstrap använder ett flexibelt gridsystem med hjälp av 12 kolumner som tillåter flera olika typer av storlekar och former av layout. Det är enkelt att skapa responsiv layout med fördefinerade klasser som ex. .col-md-6, .pull-lg-right, .hidden-md-up eller .col-auto där man kan med en speciell syntax välja exakt för vilka storlekar där klassen ska gälla. Sass variabler och mixins går att använda på samma sätt @include media-breakpoint-down(md) eller $grid-breakpoints. Systemet använder sig av 5 responsiva lager som går från den minsta telefonen upp till desktop

Foundation

Foundation är släpptes 2011 av organisationen Zurb. Precis som Bootstrap så kommer Foundation med ett funktionellt gridsystem, CSS och JavaScipt komponenter. Tidigare så ansågs Foundation vara ett av dem bästa ramverken för webbdesign speciellt då det var ett bland dem första som stödje Sass. Det senaste Foundation 6.0 släpptes 2015. Nu har man uppdelat det i "Foundation for Sites" och "Foundation for Emails" där templates och kompontenter skiljer sig beroende på miljö. Det går också att välja om man vill köra ren CSS eller Sass och även Ruby on Rails.

PureCSS

PureCSS är ett väldigt lätt och simpelt ramverk som fungerar utmärkt för exempelvis mobila applikationer då det väger endast 3.8 kb minfierat. Det bygger på Normalize.css som gör att HTML elementen framställs förenligt med moderna standarder. Även fast det är så litet så finns det ändå tillgång till komponenter för knappar, menyer och formulär och ett gridsystem. PureCSS har anpassats för att kunna överskrivas och dem uppmuntrar en att lägga på egna stilmallar. Det är också väldigt tänjbart, så om man skulle vela lägga på exempelvis bootstrap modals så är det inga problem.
